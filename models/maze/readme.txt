Maze files [.maz] readme
-------------------------------------------------------------------------------

These are not the same as the .sgv files. A maze file consists of two parts:
- the first line, describing the number of rows and columns of the maze in
  integers
- <rows> rows, with <cols> integers:
  - 1 for a wall
  - 0 for a passage
